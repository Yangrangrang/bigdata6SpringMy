스트링 라이브러리 설명

  • devtool : 리소스가 변경되면 새로고침 + liveReload(브라우저를 새로고침)
    1. 인텔리제이 settings(Preferences) > Compiler > Build Project Automatically체크
    2. 인텔리제이 settings(Preferences) > Advanced Setting > Compiler > Allow auto-make to start even if developed application is currently running 체크
    3. 크롬 웹 스토어 > liveReload 확장 프로그램 다운 -> 주소창 오른쪽 확장프로그램에 고정 추가
    4. 크롭의 개발자 도구를 열고 liveReload 확장 프로그램 실행 > 코드 변경시 브라우저 새로고침
    5. 배포할 떄는 devtool은 제외해야 한다.
  • spring-web : MVC 패턴 구현을 도와주는 spring 프레임워크 라이브러리
  • spring-thymeleaf : view를 렌더링 하는 동적파일 (템플릿 엔진)
